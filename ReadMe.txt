RobinHoodOrganisation

Overview
The RobinHoodOrganisation application is designed to manage donations, volunteers, and events for a non-profit organization. It includes features such as adding donations, listing events assigned to volunteers, and more.

Prerequisites
- Java Development Kit (JDK) 8 or higher
- Git (optional, for cloning the repository)

Getting Started

Clone the Repository
If you haven't already cloned the repository, you can do so using Git:
1. Open a terminal and run the following commands:
   git clone https://github.com/winniehhy/RobinHoodOrganisation.git
   cd RobinHoodOrganisation

2. Navigate to the src directory and compile the Java files:
   cd src/control
   javac RobinHoodOrganisation.java

3. Import hardcoded data
   Type '5' and hit enter

4. Navigate to each subsystem under control
- Donation Management
- Donation Distribution
- Event Management
- Volunteer Management
