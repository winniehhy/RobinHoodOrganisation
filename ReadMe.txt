<h1>RobinHood Organisation- Charity Management System </h1>

The RobinHoodOrganisation application is a charity management system designed to handle donations for kids. It helps manage cash, book, and toy donations, as well as volunteers and events. The application includes features such as adding donations,distribute donation, listing events assigned to volunteers, and more, making it easier for non-profit organizations to organize and track their charitable activities.

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Git (optional, for cloning the repository)
